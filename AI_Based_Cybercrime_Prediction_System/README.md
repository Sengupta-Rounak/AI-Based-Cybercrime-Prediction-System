# AI Based Cybercrime Prediction System

A portable local application combining network pre-attack forecasting, cybercrime text classification, real incident multi-label intelligence, dataset auditing, evaluation and focused downloadable reports.

## Included capabilities

- 91-feature network forecasting for 5-, 10- and 30-second horizons
- live temporal monitoring and offline traffic replay
- single-text and batch cybercrime classification
- real-data multi-label incident classification
- CSV and PCAP intake and audit
- NLP evaluation and verified-data training
- readable model validation, metrics and reports

## Model status

The network forecasting bundle remains a demonstration model and requires validation with correctly aligned real network traffic. The real incident classifier was trained on verified incident narratives and evaluated using an internal chronological holdout. It still requires independent external validation and genuine benign operational narratives before a complete threat-versus-no-threat performance claim can be made.

## Start

Extract the full folder and double-click `START.bat`. Keep the console open while the local application is running.
